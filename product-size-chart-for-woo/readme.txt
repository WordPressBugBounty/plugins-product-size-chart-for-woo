WooCommerce Size Chart lets customize and design size charts for specific products or categories, enhancing customer convenience and boosting sales.

== Description ==
WooCommerce Size Chart is a very useful and must have tool for a WooCommerce and WordPress Website. Why’s that?

If you own an online fashion store, for example, you must be wondering how products can be measured to ensure that they fit almost every customer’s size. We frequently hear that products come in S, M, L, XL… sizes, but how do these differ? WooCommerce Size Chart here is to answer and handle these problems for you.

WooCommerce Size Chart helps you to customize and design the size chart of specific products or categories, you can choose the available templates of the size chart. WooCommerce Size Chart will create conveniences for customers when shopping in your WooCommerce Store and boost your sales.

### PRODUCT SIZE CHART FOR WOO FEATURES
&#9658; **Build and customize unlimited size charts for products using the WooCommerce Size Chart plugin**
You can create a variety of size chart templates for all products on your site, the size chart displayed on product pages will provide customers with an actual choices, come from their needs.
&#9658; **Available size chart templates to choose and customize**
WooCommerce Size Chart is available with some size chart samples for common clothes for men and women, such as T-shirts, Dress, Shoes… So you can edit, customize your own in the template or duplicate templates to save time to create a new size chart.
&#9658; **Assign a size chart for single or multiple specific products and categories.**
Created size charts can be assigned for products or categories that size charts apply for directly and quickly. Assigning on which pages the size charts appear can be done directly on the size chart template. This helps shop owners save time to assign a size chart for their products and categories when they have so many products and categories on the WooCommerce store.
&#9658; **Flexible and effective size chart displays in the WooCommerce Size Chart plugin**
You can choose variously how size charts appear on the front-end. It can be:
- Popup
- Size chart tab
- Before add to cart
- Size chart shortcodes that allow showing size chart everywhere you want
&#9658; **Create and customize size charts with unlimited rows and columns.**
You can build size tables easily and quickly with a couple of clicks, the interface is completely friendly and easy to manipulate.
- Create unlimited row and a column for the Size chart table
- Quickly add multiple or remove rows and columns at the same time
- Add/Edit multiple rows and columns on the size table as per your requirements
&#9658; **Design size chart unlimited**
You can design size charts with parameters such as the rows color, text color, border color, border style,… In particular:
**Table color**: Head Background, Text Head, Even Rows Background, Even Rows Text,  Odd Rows Background, Odd Rows Text
**Border**: Horizontal Width, Vertical Width, Border Color, Horizontal Border Style, Vertical Border Style.
&#9658; **Import size chart from CSV file**
You can import a size chart from CSV file, which helps you save time to migrate your available size chart outsources to products in WooCommerce.
&#9658; **Enter size chart description**
You can enter formatted text and media into the description of your size charts. That will give a size chart with sufficient information or note for the size chart of each product.
&#9658; **Hide or show the table or content of a size chart.**
You can completely choose to public or hide size chart partially. It can be: None, Table, Image, Hide all.
&#9658; **Preview to see how size chart displays before be public**
You can visually observe size charts by the preview, update any changes in the preview mode.
&#9658; **Custom CSS**
You can add custom CSS to make your own design for a size chart in the custom CSS field of the plugin setting.
And much more things are waiting for you discover

1. Activate the plugin through the 'Plugins' menu in WordPress
